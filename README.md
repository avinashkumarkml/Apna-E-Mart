### <u>Run Locally</u>

Clone the project

```
git clone https://github.com/avinashkumarkml/Apna-E-Mart.git
```

Install npm packages

```
npm install
```

Run the surver

```
npm start
```

View in Browser

```
go to http://localhost:3000
```
